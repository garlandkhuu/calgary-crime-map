Variation 4: A static map of Calgary which displays all the LRT stations in calgary as well as every community in Calgary. The communities which are surrounding the station are filled with a colour. The colour of the community corresponds to the crime count of the community according to the legend.

The communities are supposed to be selectable in the fully interactive version and crime info for the selected community is displayed in a card on the right side above the legend. For this static implementation, the community Varsity is currently selected.
